An interactive web application built with **Python** and **Streamlit** to visualize, analyze, and explore data from [EcoBici](https://www.ecobici.cdmx.gob.mx/), the public bike-sharing system in Mexico City (CDMX).


## ✨ Features

* **Interactive Map**: View all EcoBici stations across Mexico City with real-time (or historical) availability.
* **Station Analytics**: Click on individual stations to see capacity, available bikes, and empty docks.
* **Trip Analysis**: Explore historical trip data, popular routes, and peak usage hours.
* **Filters**: Filter data by neighborhood (colonias), date ranges, or specific times of day.
* **Responsive UI**: A clean, fast, and mobile-friendly interface powered by Streamlit.

## 🛠️ Technologies Used

* **[Python](https://www.python.org/)**: Core programming language.
* **[Streamlit](https://streamlit.io/)**: For building the frontend web interface quickly and beautifully.
* **[Pandas](https://pandas.pydata.org/)**: For data manipulation and analysis.
* **[Folium / PyDeck / Plotly]**: *(Keep the ones you actually used)* For rendering interactive maps and charts.
* **EcoBici API / Open Data**: Data sourced from the [CDMX Open Data Portal](https://datos.cdmx.gob.mx/).
